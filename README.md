# HTML-Online-Resume

## Project Overview
A simple static resume website created as an HTML capstone project. It contains a main resume page, a hobbies page, and a contact page, plus an assets folder for images.

## Files
- [index.html](index.html) — Main resume page with Summary, Education, Work Experience, Skills, and Awards.
- [my hobbies.html](my%20hobbies.html) — Personal hobbies and brief descriptions.
- [contact me.html](contact%20me.html) — Contact information and links.
- [assets/](assets/) — Images and other static assets used by the site.

## How to preview
Open `index.html` directly in your browser (double-click) or serve the folder locally.

Using Python's simple HTTP server (recommended for relative links):

```bash
cd "Resume HTML Capstone project 1"
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

In VS Code, you can also use the "Live Server" extension and click "Go Live".

## How to edit
- Edit the HTML files directly in your editor.
- Replace images in the `assets/` folder and update the paths in the HTML.

## Notes
- Filenames contain spaces (e.g., `my hobbies.html`). If you rename files, update internal links accordingly.
